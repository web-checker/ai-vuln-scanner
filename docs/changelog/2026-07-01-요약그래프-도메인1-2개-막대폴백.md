## "항목별 양호 비율" — 도메인 1~2개일 때 얇은 무채색 막대로 폴백

- **증상:** 이행점검/최초진단 산출물의 `2-1. 요약결과(그래프)` 시트 1번 블록
  "항목별 양호 비율"에서, 진단 도메인이 **1개(예: 계정 관리만)** 이면 값이 100%여도
  그래프에 아무것도 안 보임(축/세로선만 남음). 표에는 `100.0%`로 값이 정상 표기됨.
- **원인:** 이 블록은 **방사형(radar) 차트**인데, 방사형은 **꼭짓점(도메인)이 최소 3개**
  있어야 면(다각형)을 그림. 도메인이 1개면 축 하나, 2개면 선 하나로 degenerate → 안 보임.
  데이터 문제가 아니라 방사형 차트의 성질.
- **조치:** 방사형은 팀원 작업이라 **3개 이상에서는 그대로 유지**(정상 데이터는 대부분 도메인 여러 개).
  1~2개일 때만 **얇은 무채색(회색) 채움 막대**로 대체.
  - 부피감이 커 방사형과 느낌이 달라지는 것을 줄이려 `gapWidth=500`(최대)로 막대를 최대한 얇게.
  - 색은 무채색 회색(`A6A6A6`) 채움 — 눈에 띄되 방사형의 절제된 느낌과 어울리도록.
- **변경 파일:** `ai_app/report.py` — `_sheet_graph()` 블록1
  - `n = len(groups)` 분기: `n >= 3` → `RadarChart`(기존 그대로),
    `n < 3` → `BarChart(type="col")` + `gapWidth=500` + 무채색 회색(`A6A6A6`) 채움.
  - y축 스케일 0~100%·`0%` 포맷·회색 축 라벨은 양쪽 공통 유지(방사형과 스케일 일치).
- **검증:** 도메인 1·2·3개 각각 `_build_workbook` → `2-1` 시트 블록1 차트 확인.
  - 1개 → `BarChart`(gapWidth 500, fill `A6A6A6`), 2개 → 동일, 3개 → `RadarChart`.
  - 세 경우 모두 .xlsx 저장(직렬화) 정상.
